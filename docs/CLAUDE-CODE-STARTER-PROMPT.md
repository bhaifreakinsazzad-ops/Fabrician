# Claude Code Starter Prompt

Use this repository as a starting scaffold only.

Your job is to transform it into the real Fabrician direction:
- kids-first futuristic fashion ecommerce
- ready-stock 0–12 months baby bodysuits as the real business
- Fabrician Studio as a preview / trial AI-assisted design experience
- signature premium mobile-first UX
- soft 3D illusion, not heavy 3D engineering

Do not preserve generic template sections just because they already exist.

## Priorities
1. Rewrite the front-facing experience first
2. Keep the current business logic only where helpful
3. Make Home, Shop, Product Detail, and Studio clearly brand-distinct
4. Treat admin pages as lower priority placeholders
5. Keep architecture clean and easy to extend

## Build order
### Phase 1
- rewrite HomePage
- rewrite ShopPage
- rewrite ProductDetailPage
- rewrite StudioPage
- improve AppHeader + MobileNav + ProductCard system

### Phase 2
- refine Cart, Checkout, OrderSuccess, TrackOrder
- refine Account basics

### Phase 3
- clean and improve Admin placeholders
- improve settings and review flow

## Hard rules
- do not convert Studio into a fully live manufacturing pipeline yet
- do not add unrelated complexity
- do not make the brand look generic
- keep the experience warm, premium, futuristic, soft, and mobile-first

## Success criteria
- homepage instantly feels branded and original
- Studio feels like a memorable signature feature
- product pages feel premium and conversion-focused
- the codebase remains maintainable
