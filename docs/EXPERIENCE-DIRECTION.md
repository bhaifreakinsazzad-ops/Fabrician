# Experience Direction for Fabrician

## The feeling
Fabrician should feel like stepping into a premium baby-fashion world from the future.
Not robotic. Not childish. Not generic.

## Core experience pillars
1. **Soft futurism**
   - subtle glow
   - airy depth
   - smooth motion
   - refined contrast

2. **Editorial commerce**
   - fewer but stronger sections
   - story-first layout
   - product detail as experience, not inventory dump

3. **Touch-friendly delight**
   - mobile-first
   - comfortable spacing
   - clear CTAs
   - premium transitions

4. **Fabrician Studio as signature layer**
   - feels magical and premium
   - clearly labeled preview / trial
   - not just a form inside a card

## Homepage should include
- strong brand opening
- immersive hero with layered composition
- signature “why Fabrician feels different” section
- product storytelling rows
- trust without generic trust-bar blandness
- Studio teaser that feels desirable
- gift / family angle section
- free-delivery Bangladesh message handled elegantly

## Product page should feel like
- boutique editorial layout
- beautiful image framing
- tactile size/color selection
- premium care and trust details
- smooth sticky purchase area on mobile

## Studio should feel like
- a guided fashion concept experience
- part inspiration tool, part design preview tool
- emotionally satisfying even before ordering exists
